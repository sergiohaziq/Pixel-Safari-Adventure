# Pixel-Safari-Adventure

A 2D jungle puzzle game built in Python using Pygame.

- About the Game
You play as an Explorer who must transport a Tiger, a Monkey, and Bananas across a muddy track in a Rescue Jeep.

-The Rules:
1. The Driver: The Explorer must be in the Jeep to drive it.
2. Space Limits: The Jeep only holds the Explorer and **one** passenger at a time.
3. Left Alone Danger:
   If left unsupervised without the Explorer:
   - The Tiger attacks the Monkey!
   - The Monkey eats the Bananas!

---------

- How to Setup and Run

⚠️ CRITICAL STEP: If you downloaded this project as a ZIP file, you must extract/unzip the folder first. Pygame cannot load the images or sounds if they stay zipped!
